# Security Reporting

Use coordinated disclosure for suspected vulnerabilities in `noxtls-rs`.

## How to report

- Email: [info@argenox.com](mailto:info@argenox.com)
- Include "Security Report: noxtls" in subject line.
- Do not publish exploit details publicly before coordination.

## What to include

- Affected version(s) and build/profile settings
- Environment details (OS, compiler, runtime constraints)
- Reproduction steps or minimal testcase
- Impact assessment (confidentiality, integrity, availability)
- Any temporary mitigation currently available

## Response process

1. Report acknowledged.
2. Triage and severity assessment.
3. Fix developed and validated.
4. Coordinated release and advisory/release-note update.
